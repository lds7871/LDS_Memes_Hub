# BotImgs - 个人表情包公开库

此项目是一个个人表情包公开库，旨在允许服务器直接通过URL调用图片，而不占用服务器资源。

## 使用方法

将此图片托管在支持直接URL访问图片的平台上（GITHUB/GITEE-同名仓库）。服务器通过URL直接引用图片。
